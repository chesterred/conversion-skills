# Conversion Skills

A multi-skill repository for common file format conversion workflows.

Included skills:

- `image-convert`
- `video-convert`
- `audio-convert`
- `document-convert`
- `data-convert`

Install and inspect:

```bash
npx skills add https://github.com/chesterred/conversion-skills --list
```

Install one skill:

```bash
npx skills add https://github.com/chesterred/conversion-skills --skill image-convert
```

Install all skills from the repo:

```bash
npx skills add https://github.com/chesterred/conversion-skills
```
